Encabezado principal con Formato

Este párrafo tiene negrita usando la etiqueta <b>, y también importancia con <strong>.

Segundo encabezado: énfasis

Aquí usamos cursiva con <i>, y énfasis semántico con <em>. También podemos subrayar con subrayado usando <u>.

Tercer encabezado: más formatos

Podemos resaltar partes del texto con <mark>, mostrar texto pequeño con este texto es más pequeño usando <small>, y marcar texto eliminado o insertado: esto está borrado y esto está insertado.

Otro encabezado con subíndice y superíndice

Ejemplo de fórmulas químicas o matemáticas: H2O (subíndice) y 23 = 8 (superíndice) con <sub> y <sup>.

Además, podemos combinar formatos: texto importante y enfatizado, o usar varias etiquetas para dar más semántica.
